DockerFlowable

Project for integrating all the flowable-components in one docker image

Open source project created when working in a flowable integration with java and the need to have all the flowable components in only one container instead of one container per functionality as designed by the flowable team. It also has a TravisCI integration so the image is build and tested automatically when a new modification is applied to the master branch.

Neulter

Art creation via Neual Network in Javascript

Project created during the HackUPC 2018. It has several use cases of how Neural Networks can be applied using Javascript in order to create art via image processing. It consists of 3 use cases, one of them changes the style of a picture by learning the changes made on other 2, the other use case is able to create a new picture using regression, and the third one is a color classification use case by using the NN.

Mushroom Analizer

Online platform to find out if a mushroom is edible or poisonous

This online solution was build as a project for a BigData subject, with it you can find the important features from a mushroom and then classifying it in the categories of edible or poisonous. Also, if no picture is available, there is a short test for the user to answer with critical questions about the mean characteristics of the mushroom. It was created with AngularJs and Node.

PasswordPurse

Secure solution for companies in order to keep secure all the passwords

Project born from the topics learned in a subject in network security as a way to keep the long passwords used by companies secure and easy to remember for the employees. It uses various security modules (RSA, secret sharing...) in order to keep the passwords secure and impossible to retrieve for third parties. It also has a TTP included so all the actions are approved by an external trustworthy entity.

Check it out
